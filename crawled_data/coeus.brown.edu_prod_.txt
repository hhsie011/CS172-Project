Coeus is the electronic Research Administration system for the University. Coeus Premium requires Java Runtime Environment (JRE) version 8 on your desktop. Install Java 8 (administrative privileges may be required). The Coeus application authenticates using your Brown User ID and Password. To become an active user, you must first complete a CAP access request form and be set up in the system with the appropriate role(s). CAP Access Form Launch Coeus Lite
