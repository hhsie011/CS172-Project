Requests submitted through the Alliance are referred to as Summit requests. Summit is available to currently registered OSU students, faculty and staff with a valid ONID account. For inormation about your ONID, please visit OSU Network ID. If you are an Ecampus student or remote Faculty or staff, please see: Using the library from a distance or check out this LibGuide. To place a Summit request, navigate to the 1Search catalog and log in using your ONID username and password to request Summit materials: If material is not available through OSU Libraries, select "Place Summit Request." Indicate if a specific volume is needed. To request multiple volumes from a multi-volume set, submit a separate Summit request for each volume. Select your pickup locaiton before submitting request. For more information about Summit, check out our Summit FAQs.
