Return material in person Return material to the book bins located outside these locations: Barnard Library Butler Library C.V. Starr East Asian Library Science & Engineering Library Users who have access to campus buildings may also return items inside at any open service desk or indoor book drop. 2. Return material by mail The Libraries cannot reimburse the cost of shipping for items returned by mail. Please send us a confirmation email once the items have been sent. Packages may be addressed to: Butler Circulation 311 Butler Library, MC 1122 535 West 114th Street New York, NY 10027 NOTE: Reserve items must be returned to the service desk or library from which they were borrowed. Items borrowed from the Law Library and Teacher’s College Library should be returned directly to those locations when possible for quickest processing. For questions not answered here, please email lio@columbia.edu.
