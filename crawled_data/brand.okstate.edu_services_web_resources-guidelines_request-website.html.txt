Once the site request has been approved by the Brand Management Web Team, IT will set up a stage site with the appropriate secure template and notify the site administrators of the stage URL. The site administrator will need to attend mandatory template training before being granted access to the site. Once the admin passes, At that time, the administrator should copy desired content to the new site. Go-live procedure When the stage site is ready to go live, please submit a ticket to helpdesk@okstate.edu to request the site to go live. The go-live process takes about 2 weeks but will include a branding and accessibility check, a security scan, and the actual go-live event. IT will redirect your URL to the new site. After the new site is live, the old site files and database will be deleted from the servers. If these files will be needed, please request a backup in your site's go live request.
