Yale Library Materials: Renew Yale Library materials using My Library Account. Interlibrary Loan: Request a renewal by emailing smlill@yale.edu. **Please note that all Borrow Direct materials must be returned. Returns If you are no longer at Yale University, we would like to ask that these materials are returned as soon as possible. Options for returns are as follows: Return to a Yale Library bookdrop (Locations are available here) Request a free shipping label (1 label for every 20lbs of books) through askyalelibrary@yale.edu. Include your name, preferred shipping address and number of labels requested. All overdue and lost item fees will be removed when materials are returned. Already Returned? If you already returned your materials, please fill out this form. We will start an extensive search of the library to try to locate these items.
