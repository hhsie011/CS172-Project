Stanford University Org Code Hierarchy This is a "tree view" of the organizational code hierarchy at Stanford University as of March 1, 2022. When the "tree view" first appears on the screen, all nodes in the hierarchy are collapsed. You can expand the hierarchy by clicking on the signs of parent nodes or collapse parent nodes by clicking on the signs. As a short-cut, you can use these buttons to expand or collapse all nodes in the hierarchy. Since there are over 2300 members of the hierarchy, it will take several seconds to perform either command. To search for an Org Code or Display Name in the hierarchy, expand all nodes of the hierarchy and use Command + F. Questions or comments about the "tree view" can be sent to the University Budget Office.
