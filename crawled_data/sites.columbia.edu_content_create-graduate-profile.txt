This template is used to create a PhD and Masters section of a site. Individual profiles automatically create a PhD and Masters landing page on the site where all the profile images and names can be seen and individual profiles accessed. Features Optional personal statement displays on top of page Tag profiles to cohort year, research interests, courses taught, review committee members and news articles Upload dissertation for users to view Automatically formats contact information, Twitter, LinkedIn, and Youtube links IMPORTANT: To create a graduate profile taxonomy terms for Cohort Year must have been previously entered. See creating taxonomies From the top navigation menu, click on Shortcuts > Add Content. From the Add Content menu, choose Graduate Profile. Enter a UNI and click Fetch from Directory to have the Columbia Directory find the profile information. Click Populate Fields to have the information automatically fill in the fields on this page. Alternatively, you can click the Continue without Directory Lookup link to enter all fields manually. Click populate fields to automatically fill out many fields in the profile. Enter or edit name information. NOTE: Displayed name is the name that appears on the page. First name and Last name are only used on the backend for sorting. Select cohort year and the type of graduate Cohort year is used for sorting the graduates on their landing page. Cohort years are created in the taxonomy section. See creating taxonomies. Enter Dissertation/Thesis title. There are additional entry fields for PhD Candidates: On the Market: If this is checked there will be a visual designation on their profile picture and PhD candidates can be filtered by On the Market. Dissertation Upload: Allows users to download and read their dissertation Review Committee Faculty Names: Hyperlinks faculty names to faculty profiles. Enter research interests. You can create a new research interest here or use a previously created research interest. Research interests entered automatically become hyperlinks, linking to the research interest page. Select Courses Taught NOTE: You can only select Courses Taught if they have been previously imported under Site Settings. See importing courses. Enter relevant new articles. NOTE: You can only select previously created News Articles. See create a news article. Enter relevant publications. NOTE: You can only select previously created publications. See create publication. Build out your page content. Enter your profile picture and all relevant fields.
