CUL - Main Content CUL - Banner Assessment Program CUL - NAV Wednesday June 1st, 2022 Navigation Behind The Scenes Assessment Program Public Site Staff Site Purpose The purpose of the Assessment Program is to be a hub for data collection and analysis for all of the CUL. The assessment program will foster a culture of assessment within CUL, to enable data-informed decision-making at all staff levels, and to promote information transparency in the organization. The Assessment Program serves library users and staff through the systematic gathering, analysis, and use of high-quality, actionable data to improve library services and resources and guide library decision-making.
