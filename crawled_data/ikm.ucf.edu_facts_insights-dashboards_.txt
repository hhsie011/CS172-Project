Access is currently limited to UCF employees with a business need. To request access to the dashboards, please submit an IKM security form. For optimal performance, the dashboards should be viewed using Chrome, Microsoft Edge, Firefox or Safari. Currently, they cannot be viewed using Internet Explorer 11. If a dashboard does not load properly upon startup, please try refreshing it (click here for instructions). This is a known issue and SAS has not released a fix for it yet. For help getting started, download the Insights Dashboards User Guide. To access to the Insights Dashboards, you must be connected either to UCF WiFi (UCF_WPA2) or the Cisco AnyConnect VPN. The dashboards below are organized by subject, and then color coded if they belong to a special grouping. Teal dashboards belong to the Co-Curricular Activities group, blue dashboards belong to the High Impact Practices group. Performance Monitoring Student Success Unit Focused Access the Dashboard on Your Mobile Device To access the dashboards on your mobile device, please download the SAS Visual Analytics app. For additional assistance, download the dashboard Mobile Setup Guide. To use the SAS VA app, you must be connected to UCF WiFi (UCF_WPA2) if you are on campus. If you are off campus, you must connect to the Cisco AnyConnect VPN app first.
