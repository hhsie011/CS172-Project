For ExamSoft exams, a personal laptop is the preferred method of testing. In addition, Mailman is beginning to support the use of iPads for ExamSoft exams. Personal laptops and iPads must meet the following requirements for ExamSoft compatibility. All laptops must be no older than approximately two years. Macintosh Requirements Personal Laptop Specification Operating System must be one of the following: (only genuine versions supported; server versions not supported) OS X 10.14.6 (Mojave) OS X 10.15 (Catalina) OS X 11 (Big Sur) *DO NOT upgrade to any newer versions of Mac OS (this includes alpha/beta versions of upcoming releases), as many required software programs will not be compatible upon newer OS availability. *Virtual Machines are NOT compatiable with some of our required software CPU = Intel processor RAM = 4 GB or greater Hard Drive = Minimum 20GB of hard drive space. Software = Internet connection for Examplify Installation, Registration, and Exam Download and Upload To check these requirements: Click on the Apple menu in the Menu bar of your Mac, and then select “About This Mac.” iPad Specifications Hardware = iPad 5+, iPad Air, iPad Mini 2+, and iPad Pro Operating System: iOS12 - iOS14. Only genuine versions of iOS are supported 500 MB of free space required to commence an exam iPad must not be Jailbroken Software = Internet connection for Examplify Installation, Registration, and Exam Download and Upload Battery: 50% to start an exam NOTE: iPad devices using Apple's new M1 chipset are not currently compatible with Examplify. Additional requirements Privacy: Strong login passwords; hardware encrypted USB keys and external drives Access to an Administrator account Examplify cannot be used on virtual operating systems Automatic backup utility for all files Windows Requirements Specifications Operating System must be one of the following: (only genuine U.S.-English, French, Portuguese, Swedish, and British versions supported) Windows 10 (32 or 64 bit) *alternative versions of Windows 10 such as RT, S, etc.. are not compatible *Other prior versions of Windows (XP, 7, 8)- are not suported *Virtual Machines are NOT compatiable with some of our required software CPU = 1.86Ghz Intel Core 2 Duo or greater List of Supported Processors Hard Drive: Minimum 20GBs of hard drive space. RAM: highest recommended for the operating system or 2GB Hard Drive: highest recommended for the operating system or 1GB of available space Software = Internet connection for Examplify Installation, Registration, and Exam Download and Upload Pre-boot authentication (PBA) Tablet Specifications Hardware = Microsoft Surface Windows 10 (32 or 64 bit) *alt versions of Windows 10 such as RT, S, etc.. are not compatible *Other tablets such as Chromebook or Netbook devices are not compatible Ram= 4gb Operating System: Windows 10 (32 or 64 bit) *alternative versions of Windows 10 labeled RT, S, etc.. are not compatible Hard Drive: highest recommended for the operating system or 1GB of available space Software = Internet connection for Examplify Installation, Registration, and Exam Download and Upload Battery: 50% to start an exam Additional Requirements Privacy: Strong login passwords; hardware encrypted USB keys and external drives. Access to an Administrator account. SofTest cannot be used on virtual operating systems. Automatic backup utility for all files. Adobe Reader (Version 9, 11, of DC) is required for exams containing PDF attachments. Screen Resolution must be 1024x768 or higher. To check these requirements on Windows 10: Right-click on the Windows button on the bottom left of the screen and then click “System.” Please check back on the CUMC IT website for any updates regarding ExamSoft Examplify requirements. Medical Center schools will be emailing more information to students in September regarding downloading and installing this software onto your computer. Software Information The Medical Center also provides software licenses without cost for many popular packages. Please see the complete list and access downloads via UNI login here. Software packages that may be required for your classwork, but are not available for online download and may require purchase include SAS, SPSS, ARC GIS, and STATA. We suggest that you confirm the software requirements with your program before contacting CUMC IT's 5help service for access.
