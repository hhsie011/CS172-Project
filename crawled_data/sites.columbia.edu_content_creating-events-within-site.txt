However you can also create events directly within the Drupal framework if needed. NOTES: The following is not supported using this method of event creation: Registering for events Creating recurring events Displaying map location for event locations You must import events from Bedework for this functionality From the Add Content menu select Event Add event information NOTE: The summary entered displays on the events listing page. It does not display on the individual event detail page. Enter event date and time NOTE: The To date field must be entered in as well. Entered featured content Use either an image, carousel or video. NOTE: Only one featured content may be used per event. Enter all the content for the event Enter all the types of content, e.g. accordions, lists, etc... for your event. NOTE: All content entered here only displays on the individual event detail page, not the event listing page. Enter location NOTE: Location entered here is unstructured content meaning it does not generate a map location. If you need this functionality you must import from Bedework. Enter a Registration Link for the event Any URL can be set for Registration, e.g. this can be an eventbright URL. It will render as a Register button. Enter categories for event Filtered listings for events based on categories can be created. See Finding TermID.
