Featured Resources Secure Data Enclave Secure Data Enclave (SDE) provides Columbia researchers with a secure, remotely accessible, virtual Windows 10 desktop environment to store and collaboratively analyze PII, PHI, and RHI data as an alternative to traditional cold room computing environments. LabArchives Columbia University provides an electronic research notebook service for researchers, instructors, and students. This service helps organize and store laboratory data, provides information sharing, and enables collaboration, all with automated backups and a comprehensive audit trail. ReaDI Program The Research and Data Integrity (ReaDI) program is designed to enhance data management and research integrity at Columbia University. Code Ocean Code Ocean allows researchers to create, share, collaborate on, and run code for private projects or to publish in academic journals. Columbia University is piloting Code Ocean. This program provides expanded storage capacity and compute time beyond the free tier among other benefits.
